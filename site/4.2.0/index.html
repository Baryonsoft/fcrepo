About Fedora Commons 4

Parent project for Fedora Commons 4

Project Modules

This project has declared the following modules:

Name Description
Fedora Repository BOMs A parent project for including Fedora Repository BOM modules.
Fedora Repository Minter Module The Fedora Commons repository Minter module: Provides repository identifier and path services.
Fedora Repository Kernel API Fedora Repository Kernel API, containing types and constants for working with Fedora 4
Fedora Repository Kernel Implementation The Fedora Commons repository kernel: Provides the basic abstractions at the heart of the Fedora information architecture and machinery for manipulating them.
Fedora Repository Deployable Web Application The Fedora web application
Fedora Repository JMS Module The Fedora Commons repository JMS module: Provides repository event publication via Java Message Service.
Fedora Repository HTTP Commons Module The Fedora Commons repository HTTP commons module: Provides HTTP (JAX-RS) resources and helpers for use in assembling HTTP APIs over the Fedora Commons repository framework.
Fedora Repository HTTP API Module The Fedora Commons repository HTTP API: Provides a RESTful HTTP API to interact with the Fedora Commons repository.
Fedora Repository Metrics Module The Fedora Commons repository metrics module: Provides internal repository metrics reporting.
Fedora Repository Object Serialization Module Fedora serialization module
Fedora Repository RDF Transformations Module The Fedora Commons repository transformations module: Provides facilities to use SPARQL or LDPath to tranform repository RDF into desired forms.
Fedora Repository Authorization Commons Module Provides unified policy enforcement points for various access control implementations.
Fedora Repository FileSystem Connector Module The Fedora Commons repository filesystem connector module: Provides repository projection over hierarchical files/directories on the filesystem.
Fedora Repository Configurations Module The Fedora Commons repository configurations module: Provides configuration resources that are used in integration testing and deployments.
Fedora Repository LDP integration tests Module Packaging the LDP integration test suite for fcrepo4
Fedora Repository RDF integration tests Module Fedora RDF integration scenarios
